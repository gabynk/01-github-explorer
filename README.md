Ferramentas utilizados:

- [Sass](https://sass-lang.com/)

## 🎲 Como rodar?

```bash
# Instale as dependências
yarn add 
# desenvolvimento
yarn dev 
# produção
yarn build 
# acesse a aplicação por
http://localhost:8080/
```
